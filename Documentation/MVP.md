### Persona principal: Usuário (leitor)

### Objetivo: O leitor ter controle dos livros que já completou a leitura.

### Funcionalidades MVP (máx. 6):
1. Adicionar livros;
2. Mostrar livros adicionados;
3. Excluir livros;
4. Colocar data a qual iniciou e terminou livro;
5. Pesquisar livros adicionados por ID e nome;

### Fora do MVP (até verção 2):
1. Adicionar IA para orientar opções de leitura com base nos livros adicionados;
2. IA que converse sobre o livro adicionado
3. Correção de livros digitados de forma errônea;


### Prazo fechado: 3 semanas;
•
Critérios de pronto: teste + docs + erro + validação;
