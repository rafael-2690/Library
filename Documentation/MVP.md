### Persona principal: Usuário (leitor)

### Objetivo: Receber indicação de livros, com base nos livros de entrada.

### Funcionalidades MVP (máx. 6):
1. Adicionar livros;
2. Mostrar livros adicionados;
3. Excluir livros;
4. Indicar livros;
5. Correção de livros digitados de forma errônea;

### Fora do MVP (até verção 2):
1. Guardar as informações em um banco de dados;
2. Receber informações sobre os livros adicionado ou indicados;


### Prazo fechado: 3 semanas;
•
Critérios de pronto: teste + docs + erro + validação;
